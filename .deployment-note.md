# 部署说明

本项目需要服务端代理 TikTok API，不能直接部署为 GitHub Pages 静态站。

可用部署方式：

- Render / Railway / Fly.io / Vercel Serverless，配置环境变量后运行 `npm start`。
- 本地运行：`npm start`，访问 `http://127.0.0.1:8787/`。

必须配置：

- `TIKTOK_ACCESS_TOKEN`
- `TIKTOK_ADVERTISER_ID`

可选配置：

- `TIKTOK_ADVERTISER_NAME`

不要把真实 Token 提交到 GitHub。
